# Badges GUI v2 - Complete Implementation Plan

## Overview

This plan covers:
1. NIP-58 Extension implementation (kinds 30058, 30059)
2. Five new major features for Badges GUI

---

## New Features Summary

| # | Feature | Description |
|---|---------|-------------|
| 1 | **Surf Tab** | Browse all badges on Nostr, view details, request badges |
| 2 | **User Search + LookupUser** | Search users, view profile + badges in shared component |
| 3 | **Postbox** | Notification inbox for badge requests with award/deny actions |
| 4 | **My Awarded Badges** | View recipients of badges I created, grouped by badge |
| 5 | **LookupUser Integration** | Click any user anywhere to see their profile + badges |

## Feature 5: LookupUser Integration

### Purpose
Clicking on any user anywhere opens the shared LookupUser component.

### Integration Points

| Location | Trigger | Action |
|----------|---------|--------|
| Surf - Badge holders | Click holder avatar | Open LookupUser modal |
| Surf - Badge issuer | Click issuer name | Open LookupUser modal |
| Search results | Click search result | Open LookupUser modal |
| Postbox - Requester | Click requester | Open LookupUser modal |
| My Awards - Recipients | Click recipient | Open LookupUser modal |
| Badge Detail - Holders | Click holder | Open LookupUser modal |
| LookupUser - Badges | Click badge | Open Badge Detail modal |

### Implementation Pattern

```vue
<!-- Any component that shows a user -->
<template>
  <div @click="openUserLookup(user.pubkey)">
    <UserAvatar :pubkey="user.pubkey" />
    <span>{{ user.name }}</span>
  </div>
</template>

<script setup>
import { useUIStore } from '@/stores/ui'

const ui = useUIStore()

function openUserLookup(pubkey) {
  ui.openLookupUser(pubkey)
}
</script>
```

### UI Store Extension

```javascript
// frontend/src/stores/ui.js

export const useUIStore = defineStore('ui', {
  state: () => ({
    // Existing
    toasts: [],
    loading: false,

    // New - Modal states
    lookupUserModal: {
      isOpen: false,
      pubkey: null
    },
    badgeDetailModal: {
      isOpen: false,
      badgeAtag: null
    }
  }),

  actions: {
    openLookupUser(pubkey) {
      this.lookupUserModal = { isOpen: true, pubkey }
    },
    closeLookupUser() {
      this.lookupUserModal = { isOpen: false, pubkey: null }
    },
    openBadgeDetail(badgeAtag) {
      this.badgeDetailModal = { isOpen: true, badgeAtag }
    },
    closeBadgeDetail() {
      this.badgeDetailModal = { isOpen: false, badgeAtag: null }
    }
  }
})
```

### Global Modal Container

```vue
<!-- App.vue or MainLayout.vue -->
<template>
  <div id="app">
    <router-view />

    <!-- Global modals -->
    <LookupUserModal
      v-if="ui.lookupUserModal.isOpen"
      :pubkey="ui.lookupUserModal.pubkey"
      @close="ui.closeLookupUser()"
      @badge-click="ui.openBadgeDetail($event)"
    />

    <BadgeDetailModal
      v-if="ui.badgeDetailModal.isOpen"
      :badge-a-tag="ui.badgeDetailModal.badgeAtag"
      @close="ui.closeBadgeDetail()"
      @user-click="ui.openLookupUser($event)"
    />
  </div>
</template>
```
